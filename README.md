# Exercise 1

## Getting Started:

Clone the repo: 

`git clone https://github.com/toose/ba-exercise`

Change directory to cloned repo:

`cd /path/to/ba-exercise`

Build/run containers from docker-compose.yml:

`docker-compose up -d --build`
